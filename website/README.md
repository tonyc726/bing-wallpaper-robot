# Bing Wallpaper Frontend

基于 React + TypeScript + Vite 构建的现代化 Bing 壁纸展示与管理系统。

## 🌟 主要特性

- **海量壁纸展示**: 采用分块增量加载技术（Chunked Loading），支持流畅浏览数千张历史壁纸数据，首屏不卡顿。
- **多级混合缓存架构 (v4.0)**:
  - 内置基于 IndexedDB 的多级缓存引擎（内存热级、LRU 回收、本地持久化存储）。
  - 支持弱网环境及**完全离线**情况下的浏览访问。
- **多维度检索与过滤**:
  - 顶部全局**全文搜索**（支持标题、配文、关键词匹配）。
  - 支持按**年份快捷直达**（如 2025、2024）。
  - 支持按时间正序/倒序排列。
  - “仅显示收藏”独立过滤视图。
- **沉浸式画廊**:
  - 支持全屏深色沉浸式高清大图查看。
  - 完整展示拍摄地、版权声明、日期和每日俳句/配文等元数据。
  - 键盘左右键或点击箭头快速翻页。
- **个性化操作**:
  - **❤️ 本地收藏**: 喜欢的壁纸可加入本地收藏夹，偏好持久化。
  - **⬇️ 原图下载**: 提供一键提取并下载无水印高清原图。
  - **🌓 主题切换**: 支持深色/浅色模式平滑切换，自动保存用户偏好。
- **PWA (渐进式 Web 应用)**: 支持离线缓存资源，可添加到桌面或手机主屏幕，体验媲美原生 App。
- **专属开发者面板**:内置 IndexedDB 性能监控控制台（快捷键 `Ctrl/Cmd + D` 或右上角图标），实时监控数据库命中率、读写次数、网络下载量及操作日志。

## 🚀 核心技术栈

- **框架**: React 18, TypeScript, Vite
- **UI 组件库**: Material-UI (@mui/material), @emotion/react
- **状态与路由**: React Router v7, nuqs (URL 参数状态同步)
- **数据流层**: 原生 IndexedDB 通道与自定义调度器
- **时间处理**: date-fns
