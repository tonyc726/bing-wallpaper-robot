# 拾影阁 (Lumina Pavilion) — 前端应用

轻量级、高颜值的现代化 Web 在线壁纸博物馆，基于 React + TypeScript + Vite 构建。

## 🌟 主要特性

- **海量馆藏展示**: 采用分块增量加载技术（Chunked Loading），支持流畅浏览数千张历史壁纸数据，首屏不卡顿。
- **多级混合缓存架构 (v4.0)**:
  - 内置基于 IndexedDB 的多级缓存引擎（内存热级、LRU 回收、本地持久化存储）。
  - 支持弱网环境及**完全离线**情况下的浏览访问。
- **多维检索与探索**:
  - 顶部全局**全文搜索**（支持标题、配文、关键词匹配）。
  - 支持按**年份快捷直达**（如 2025、2024）。
  - 支持按时间正序/倒序排列。
  - 支持按**主色调**（颜色分类）视觉检索。
- **沉浸式画廊**:
  - 支持全屏深色沉浸式高清大图查看。
  - 完整展示拍摄地、版权声明、日期和每日俳句/配文等元数据。
  - 键盘左右键或点击箭头快速翻页。
- **极简交互**:
  - **⬇️ 一键下载**: 提供直观、无广告的高清原图下载途径。
  - **🌓 主题切换**: 支持深色/浅色模式平滑切换（默认深色），自动保存用户偏好。
  - **🔗 分享直达**: 每张壁纸均可生成专属 Deep Link 直达链接。
- **PWA (渐进式 Web 应用)**: 支持离线缓存资源，可添加到桌面或手机主屏幕，体验媲美原生 App。

## 🚀 核心技术栈

- **框架**: React 18, TypeScript, Vite
- **UI 组件库**: Material-UI (@mui/material), @emotion/react
- **状态与路由**: nuqs (URL 参数状态同步)
- **数据流层**: 原生 IndexedDB 通道与自定义调度器
- **时间处理**: date-fns
