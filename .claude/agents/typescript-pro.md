## Type Conventions

### Interface vs Type

| Scenario | Use | Example |
| --- | --- | --- |
| Object structures | `interface` | `interface ReportItem { uid: string }` |
| Unions | `type` | `type Status = 'active' \| 'inactive'` |
| Intersections | `type` | `type Detailed = Report & { extra: string }` |
| Function types | `type` | `type Handler = (data: T) => void` |
| Utility types | `type` | `type ReportList = ReportItem[]` |
| Mapped types | `type` | `type Readonly<T> = { readonly [K in keyof T]: T[K] }` |

### No `any`

```typescript
// ❌ Forbidden
const data: any = fetchData();

// ✅ Required: use `unknown` with type guards
const data: unknown = fetchData();
if (typeof data === 'string') console.log(data.toUpperCase());
```

### Naming

| Category            | Convention        | Example                         |
| ------------------- | ----------------- | ------------------------------- |
| Types/Interfaces    | PascalCase        | `ReportItem`, `QueryParams`     |
| Variables/Functions | camelCase         | `fetchReportList`, `reportData` |
| Constants           | UPPER_SNAKE_CASE  | `API_CODE`, `DEFAULT_PAGE_SIZE` |
| Generics            | Single uppercase  | `T`, `K extends keyof T`        |
| Boolean props       | is/has/can prefix | `isLoading`, `hasNextPage`      |

### Component Props

Interface MUST be named `[ComponentName]Props`.

```typescript
// ✅ Correct
interface AlgorithmConfigPanelProps {
  /** Algorithm type identifier */
  algorithmType: 'peak' | 'baseline';
  /** Chart width in pixels */
  width: number;
  /** Loading state flag */
  isLoading: boolean;
  /** Unsaved changes flag */
  hasUnsavedChanges: boolean;
  /** Config change callback */
  onConfigChange: (config: AlgorithmConfig) => void;
  /** Optional CSS class */
  className?: string;
}

// ❌ Wrong
interface Props {
  // Should be AlgorithmConfigPanelProps
  type: string;
  loading: boolean; // Should be isLoading
  onChange: Function; // Should be onConfigChange with type
}
```

**Props Rules**:

- Interface: `[ComponentName]Props`
- All properties: JSDoc comment `/** description */`
- Callbacks: `on*` prefix
- Booleans: `is*` / `has*` / `can*` prefix

## Global API Types

Location: **`src/services/typings.d.ts`**

```typescript
export type ApiResponse<T> = {
  data: T;
  code: number;
  message?: string;
  targetService?: string;
  detailMsg?: string;
};

export type PaginatedData<T> = {
  list: T[];
  total: number;
  pageNum: number;
  pageSize: number;
  hasNext: boolean;
};

export type Nullable<T> = T | null;
export type MaybeArray<T> = T | T[];
```

**Reuse Principle**: Import from `@/services/typings`, never redefine.

```typescript
// ✅ Correct
import type { ApiResponse, PaginatedData } from '@/services/typings';
export type QueryReportResponse = ApiResponse<PaginatedData<ReportItem>>;

// ❌ Wrong
interface MyResponse<T> {
  data: T;
  code: number;
} // Duplicate!
```

### API Function ESDoc

```typescript
/**
 * Fetch report list by sampling point and time range
 * @description Retrieves chromatography reports for a sampling point
 * @method GET
 * @path /cda/v1/qlabCdaReport/list/{pickSiteUid}
 * @tag Chromatography Report
 * @param {object} params - Query parameters
 * @param {string} params.startTime - Start time (ISO format)
 * @param {string} params.endTime - End time (ISO format)
 * @param {string} pickSiteUid - Sampling point UID
 * @returns {Promise<ReportListResponse>} Report list
 */
export async function fetchReportList(
  params: { startTime: string; endTime: string },
  pickSiteUid: string,
): Promise<ReportListResponse> {
  return request<ReportListResponse>(
    `/cda/v1/qlabCdaReport/list/${pickSiteUid}`,
    { method: 'GET', params },
  );
}
```

### No Magic Numbers

⚠️ **ALWAYS import status codes from `src/services/code.ts`**

```typescript
// src/services/code.ts
export const API_CODE = {
  SUCCESS: 100000000,
  UNAUTHORIZED: 100000001,
  FORBIDDEN: 100000002,
  NOT_FOUND: 100000003,
  SERVER_ERROR: 100000500,
} as const;

// ✅ Correct: Import and use constants
import { API_CODE } from '@/services/code';

request.interceptors.response.use(
  (response) => {
    if (response.data.code === API_CODE.SUCCESS) {
      return response.data.data;
    }
    return Promise.reject(response.data);
  },
  (error) => {
    if (error.response?.data?.code === API_CODE.UNAUTHORIZED) {
      redirectToLogin();
    }
    return Promise.reject(error);
  },
);

// ❌ Forbidden: Hardcoded magic numbers
if (response.code === 100000000) {
  // Magic number!
  return response.data;
}

const errorMap = {
  100000001: 'Unauthorized', // Magic number!
};
```

## Type Guards

```typescript
// ✅ Type guard function
function isReportItem(value: unknown): value is ReportItem {
  return (
    typeof value === 'object' &&
    value !== null &&
    'uid' in value &&
    typeof (value as Record<string, unknown>).uid === 'string'
  );
}

// ✅ Discriminated union
type APIResult =
  | { success: true; data: ReportItem }
  | { success: false; error: string };

function handleResult(result: APIResult) {
  if (result.success) {
    console.log(result.data); // Narrowed to success branch
  } else {
    console.error(result.error); // Narrowed to error branch
  }
}
```
